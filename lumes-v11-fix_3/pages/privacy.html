2. Données collectées

Nous collectons uniquement les données nécessaires à nos services :

  • Données de compte : nom, email, mot de passe (hashé — jamais en clair)
  • Données de commande : adresse de livraison, téléphone, contenu du panier
  • Données de paiement : traitées directement par Stripe — nous ne stockons aucune donnée bancaire
  • Données de navigation : pages visitées, durée de session (via Google Analytics — anonymisé)

3. Finalités et bases légales

  • Traitement des commandes — base légale : exécution du contrat
  • Gestion du compte client — base légale : exécution du contrat
  • Envoi d'emails transactionnels (confirmation, suivi) — base légale : exécution du contrat
  • Envoi de newsletters — base légale : consentement (opt-in)
  • Amélioration du site (analytics) — base légale : intérêt légitime / consentement
  • Obligations légales (comptabilité, fiscalité) — base légale : obligation légale

4. Destinataires des données

Vos données sont transmises uniquement aux partenaires strictement nécessaires :

  • Stripe — paiement sécurisé (certifié PCI DSS niveau 1)
  • Transporteurs — pour la livraison (nom, adresse, téléphone uniquement)
  • Google Analytics — données de navigation anonymisées
  • Netlify — hébergement du site

Nous ne vendons jamais vos données à des tiers.

5. Durée de conservation

  • Données de compte : durée de vie du compte + 3 ans après clôture
  • Données de commande : 10 ans (obligation comptable)
  • Données de navigation : 14 mois (paramétrage Google Analytics)
  • Emails marketing : jusqu'au désabonnement ou 3 ans sans activité

7. Sécurité

Nous mettons en œuvre les mesures techniques et organisationnelles appropriées pour protéger vos données :

  • Chiffrement SSL/TLS de toutes les communications
  • Mots de passe hashés (SHA-256 + sel)
  • Accès aux données restreint aux personnes habilitées
  • Données bancaires traitées exclusivement par Stripe (jamais stockées chez nous)

8. Transferts hors UE

Certains de nos prestataires (Netlify, Stripe, Google) sont établis aux États-Unis. Ces transferts sont encadrés par des garanties appropriées (clauses contractuelles types de la Commission européenne, certification Data Privacy Framework).
